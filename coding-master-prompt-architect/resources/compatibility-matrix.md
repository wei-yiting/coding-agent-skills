# Compatibility Matrix: Integration Guidelines

## Overview

This document provides systematic guidelines for integrating different analytical frameworks, meta-prompt sections, and specialized applications. The compatibility matrix ensures optimal framework selection and combination for enhanced analytical performance while avoiding conflicts and redundancies.

## Quick Reference: Compatibility Selection Guide

### By Analytical Complexity

**Simple Analysis (Low Complexity):**
- Use foundational frameworks with high compatibility sections
- Avoid elite frameworks and challenging combinations
- Focus on CO-STAR, CLEAR, or RACE with standard thinking systems

**Moderate Analysis (Medium Complexity):**
- Use advanced frameworks with moderate compatibility management
- Consider hybrid combinations with careful integration
- Focus on SMART, FOCUS, or DRIVE with specialized thinking systems

**Complex Analysis (High Complexity):**
- Use elite frameworks with sophisticated compatibility management
- Leverage challenging combinations with expert integration
- Focus on MASTERY, EXCELLENCE, or TRANSFORMATION with advanced thinking systems

### By Time Constraints

**Rapid Analysis (< 2 hours):**
- Prioritize high-efficiency combinations
- Use RACE, TAG, or Short-Form R+R frameworks
- Avoid complex integration and challenging combinations

**Standard Analysis (2-8 hours):**
- Balance efficiency and effectiveness
- Use moderate compatibility combinations
- Consider CO-STAR, SMART, or FOCUS with appropriate thinking systems

**Comprehensive Analysis (> 8 hours):**
- Prioritize effectiveness over efficiency
- Use complex integration and challenging combinations
- Consider EXCELLENCE, BREAKTHROUGH, or Long-Form R+R frameworks

### By Quality Requirements

**Standard Quality:**
- Use high compatibility combinations
- Focus on proven framework combinations
- Minimize integration complexity and potential conflicts

**High Quality:**
- Use moderate compatibility with quality assurance integration
- Include validation protocols and error detection patterns
- Balance analytical capability with reliability requirements

**Maximum Quality:**
- Use challenging combinations with comprehensive quality assurance
- Include multiple validation approaches and continuous improvement
- Prioritize analytical excellence over efficiency or simplicity

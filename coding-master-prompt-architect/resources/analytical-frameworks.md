# Analytical Frameworks: 67 Foundational Cognitive Components

## Overview

This document contains the complete catalog of 67 foundational cognitive components that form the analytical foundation for hybrid meta-prompt construction. These components are organized into two primary sections: Thinking Systems Catalog (55 Systems) and Foundational Patterns Catalog (16 Patterns).
